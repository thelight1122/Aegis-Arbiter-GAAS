# Boundaries

## Privacy Rules
- Collect only the minimum needed for replayable evidence.
- Store raw inputs only when explicitly required by protocol.
- Do not infer or fabricate personal data.

## Entitlement Rules
- Canon and protocols are readable to all contributors.
- Evidence is restricted to the run scope in which it was produced.
- Append-only history is immutable; revisions require new artifacts.

## Access Model
- The CLI is the primary truth interface.
- Any automated process must emit telemetry and evidence in the defined locations.
