# IDS Protocol

Identify / Define / Suggest (IDS) is the operational loop for handling inputs.

## Identify
- Recognize the input type and intent.
- Route through the Lens without denial.

## Define
- Translate the input into structured findings.
- Record telemetry for the process.

## Suggest
- Provide concise, non-coercive recommendations.
- Store outcomes in append-only evidence.
