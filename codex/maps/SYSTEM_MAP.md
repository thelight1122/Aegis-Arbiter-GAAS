# System Map

## Layers
- Canon: locked truths and definitions.
- Protocols: operational rules (SSSP, IDS, LINT).
- Artifacts: snapshots, decisions, schemas.
- Runtime: Engine, Lens, Arbiter, Peer Tensor, Telemetry.
- Evidence: append-only run outputs.

## Connections
Lens routes inputs to Arbiter. Arbiter emits findings and IDS suggestions. Peer Tensor stores alignment deltas. Telemetry records the run as JSONL.
